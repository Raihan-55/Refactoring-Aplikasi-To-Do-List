## Deskripsi Proyek

Proyek ini merupakan implementasi refactoring sederhana dari aplikasi To-Do List berbasis Python. 
Struktur proyek telah dipisahkan ke dalam beberapa layer untuk memisahkan tanggung jawab, yaitu:

- **Presentation Layer**: Menangani interaksi dengan pengguna (input dan output).
- **Business Layer**: Menangani logika aplikasi.
- **Data Layer**: Menangani penyimpanan dan pengelolaan data.

Pemisahan ini bertujuan agar kode lebih terstruktur, mudah dikembangkan, dan lebih mudah diuji.
